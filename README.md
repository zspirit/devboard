# devboard

A developer dashboard that displays project metrics, commit activity, and deployment history. Built with React and Vite.

## Features

- Overview page with stat cards, charts, and activity feed
- Commit history with branch and author info
- Deployment log with status tracking
- Responsive layout with sidebar navigation
- Dark theme inspired by GitHub

## Tech Stack

- **React 18** — UI library
- **Vite** — build tool
- **Recharts** — charts and data visualization

## Getting Started

```bash
git clone https://github.com/zspirit/devboard.git
cd devboard
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.
